# 东南大学本科毕业设计（论文）LaTeX模版
本项目包含东南大学本科毕业设计（论文）LaTeX模版（2019版规范）。

## 依赖环境

- TeX Live 2020及以上，或MikTeX2020及以上

  

## 使用方法
- Windows下：

  1. 运行

     ```cmd
     xelatex seuthesis-2019.ins
     ```

     以生成cls、cfg文件。

  2. 运行

     ```cmd
     ./sample_thesis.bat
     ```

     以编译样例论文。

- Mac下：

  1. 运行

     ```shell
     xelatex seuthesis-2019.ins
     ```

     以生成cls、cfg文件。

  2. 运行

     ```shell
     ./sample_thesis.sh
     ```

     以编译样例论文。
